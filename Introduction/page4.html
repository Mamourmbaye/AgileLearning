Key Concepts

Item: Component or document used to manufacture a product.
        Example: Capacitor, Solder Paste, pens, glue.

BOM: List of the necessary elements to manufacture a product.
        Example: Components, quantities, reference designators, documents, tooling…

Object: All elements that can be created and managed within Agile.
        Example: Sandbox, RFQ, Change order, Deviations…
